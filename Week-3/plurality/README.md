# Plurality (C)

A program that simulates a plurality vote (first-past-the-post) election between any number of candidates.

## How it works

- Candidates are passed as command-line arguments
- Takes a vote input from each voter
- Declares the candidate with the most votes as the winner, or all tied candidates in case of a tie

## Compile & Run

```
make plurality
./plurality Alice Bob Charlie
```
