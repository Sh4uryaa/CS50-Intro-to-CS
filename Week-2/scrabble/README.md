# Scrabble (C)

This program determines the winner of a Scrabble round between two players based on the point values of their submitted words.

## How it works

- Takes a word input from each player
- Calculates the Scrabble score for each word using standard letter point values
- Declares the higher-scoring player as the winner, or a tie if scores are equal

## Compile & Run
```
gcc scrabble.c -o scrabble
./scrabble
```
